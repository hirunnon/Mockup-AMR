Project "Yens" Smart Warehouse

ระบบบริหารคลังสินค้ากึ่งอัตโนมัติ ด้วย Odoo Enterprise และ Industrial IoT

1. ภาพรวมโครงสร้างระบบ (System Architecture)

การเชื่อมต่อระหว่างระบบ ERP บน Cloud และอุปกรณ์ควบคุมในคลังสินค้า ผ่านเทคโนโลยีที่ปลอดภัยและเสถียร

แผนภาพแสดงการเชื่อมต่อ: Odoo ส่งคำสั่งผ่าน Tunnel มายัง Raspberry Pi เพื่อควบคุม Hardware

  • Core System: Odoo 17 Enterprise (On Cloud)
  • Connectivity: Cloudflare Tunnel (Secure Link) + MQTT Protocol
  • Controller: Raspberry Pi 5 (8GB/16GB) + NVMe SSD

2. ขั้นตอนการทำงาน (Operational Workflow)

1
Order & Payment (จุดเริ่มต้น)

พนักงานเปิด Invoice ในระบบ Odoo ระบบจะสร้าง Dynamic QR Code บนใบแจ้งหนี้อัตโนมัติ เมื่อบัญชียืนยันยอดเงิน (Reconcile) สถานะคำสั่งซื้อจะเปลี่ยนเป็น "Ready to Pick"

2
Command Dispatch (ส่งคำสั่ง)

Odoo ส่งคำสั่งผ่าน MQTT วิ่งผ่าน Cloudflare Tunnel ลงมาที่โกดังขอนแก่น Raspberry Pi 5 รับคำสั่งและประมวลผลด้วย Node-RED ทันที (Latency ต่ำมาก)

3
Execution (จ่ายสินค้าอัตโนมัติ)

Node-RED สั่งงาน Hardware ตามประเภทสินค้า:

  • สินค้าชิ้นเล็ก: สั่ง Solenoid เปิดล็อก -> สินค้าไหลลงราง (Gravity Chute)
  • สินค้าหนัก: สั่ง Relay ขับสายพาน MDR -> ลำเลียงสินค้าออกมา
4
Packing & Validation (จบงาน)

สินค้าไหลมารวมที่จุดแพ็ค หน้าจอ Kiosk แสดงรายการ พนักงานตรวจสอบและ สแกนบัตร (Badge) เพื่อยืนยัน ระบบส่ง XML-RPC กลับไปตัดสต็อกใน Odoo ทันที

3. สเปกอุปกรณ์หลัก (Hardware Specifications)

Main Controller

Raspberry Pi 5

  • RAM: 16GB (High Performance)
  • Storage: 256GB NVMe SSD
  • Cooling: Active Cooler

Connectivity

Network & IoT

  • Gigabit Ethernet (Static IP)
  • Cloudflare Tunnel (Zero Trust)
  • MQTT Broker (Mosquitto)

Automation Parts

Control Modules

  • 8-Channel Relay Module
  • 12V/24V Solenoid Locks
  • Infrared Break Beam Sensors
